# Flappy Bird
Техническое задание
1. Птичка должна прыгать через препятсвия (зеленые трубы).
2. При столконовении с полом или трубой игроку должно высветиться сообщение о поражении.
# План
1. Пройти курс обучения Python'у.
2. ПРОГРАММИСТУ: Написать код для игры и протестировать его.
3. МЕНЕДЖЕРУ И ТЕСТИРОВЩИКУ: Сделать красивую презентацию для игры. Если программист допустил ошибки в коде, найти и исправить их. Сделать повторный тест игры.
